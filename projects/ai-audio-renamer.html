AI Audio Renamer.

A semantic file intelligence tool that analyses audio content and renames files with meaningful, taxonomy-based labels — turning chaotic sample libraries into instantly searchable collections.

Problem

Music producers accumulate thousands of audio files with meaningless names. During a creative session, searching for the right sample breaks flow. The conventional solution — manual tagging — doesn't scale. The right approach is to let the audio content itself determine the metadata.

How It Works

The tool processes audio files in batch through OpenAI Whisper for content analysis, then uses GPT-4 (or Ollama for fully offline operation) to generate structured labels following a consistent taxonomy: Genre → Instrument → Mood → BPM → Key. Drop a folder, get it renamed in minutes.

Design Decisions

  • Local-first option: Ollama support ensures sensitive audio content never leaves the machine.
  • Batch-native: Designed for folder-level operations, not individual files. Producers work in projects, not clips.
  • Taxonomy consistency: Rigid naming convention ensures every file in a library follows the same structure regardless of when it was processed.

Role: Sole builder

Stack: Python, OpenAI Whisper, GPT-4, Ollama
