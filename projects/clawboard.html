ClawBoard.

A high-end control tower for the autonomous agent era.

The Vision

As autonomous systems handle more of our workflows, there is a need for a unified control interface. ClawBoard is designed to monitor agent health, task status, and operational costs across multiple AI systems — Claude, n8n workflows, and custom automation scripts — from a single glassmorphic dashboard.

Design Language

The UI follows a strict Apple-esque minimalist aesthetic. High-contrast grids, glassmorphism, and bold typography present complex system data in a way that feels premium and intuitive. Every element is designed to surface signal, not noise.

Core Modules

  • Task Kanban: Real-time sync with Obsidian project boards and Asana.
  • Cost Watchdog: Token usage monitoring across LLM providers (Gemini, OpenAI, Anthropic).
  • Agent Status: Health and throughput visualisation for Clawy, n8n, and MCP servers.
  • Workflow Inspector: Drill into individual n8n execution logs and error states.

Role: Product Designer & Builder
