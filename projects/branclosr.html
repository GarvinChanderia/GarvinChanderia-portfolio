B2C SaaS

BranClosr.

Dual-sided marketplace cutting brand–influencer deal time from 30 days to under 48 hours.

The Problem

Influencer marketing deals take 2–4 weeks to close. Brands and creators lose momentum in endless email chains, negotiation back-and-forth, and contract ambiguity. BranClosr compresses the entire cycle — discovery, negotiation, contract, payment — into a single platform targeting sub-48-hour campaign cycles for D2C brands and mid-tier influencers (10K–500K followers) in the Indian market.

Technical Architecture

Full-stack PWA built with Next.js 16 App Router (RSC + client components), TypeScript in strict mode, and Supabase/PostgreSQL with row-level security. State management via React Hooks, URL search params, and Zustand stores. Forms powered by React Hook Form + Zod validation. Data fetching through TanStack React Query. Styled with Tailwind CSS + shadcn/ui component patterns.

Frontend: Next.js 16.1.6, TypeScript 5.x, Tailwind CSS 4.x, Lucide icons

State: Zustand, TanStack React Query 5.x, URL search params

Forms: React Hook Form 7.x + @hookform/resolvers + Zod 4.x

Backend: Supabase (PostgreSQL 15+, PostgREST, Realtime, Auth, Storage)

Auth: Supabase Auth (email/password + Google OAuth) via @supabase/ssr

Charts: Recharts 3.x

Testing: Vitest + React Testing Library + JSDOM

Hosting: Vercel (auto-deploy from GitHub main)

What's Built

56 TypeScript/TSX source files across a clean modular structure: public landing page with SEO optimisation and OpenGraph/JSON-LD, auth flows (login, register, OAuth callback), role-aware dashboard with campaigns, deals, deliverables, notifications, and profile pages. Components split into layout (header, sidebar, mobile nav, role guard), domain-specific (campaign cards, deal timelines, status badges), and shared (confirm dialogs, error boundaries, loading skeletons).

Key Features

  • Deal Rooms: Real-time negotiation spaces with structured offer/counter-offer flows and status propagation.
  • Smart Matching: AI-driven brand–influencer pairing based on niche, engagement, and budget fit.
  • Contract Engine: Auto-generated agreements with configurable terms.
  • Deliverables Tracking: Grouped by campaign with full lifecycle management.
  • Notifications: Mark-as-read, real-time updates, grouped by type.
