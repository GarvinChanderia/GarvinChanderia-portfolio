Infrastructure

Self-Hosted NAS.

Repurposed laptop hardware running Ubuntu Server with CasaOS — providing local file storage, media streaming, and on-premise AI model hosting at zero recurring cost.

Motivation

Cloud storage subscriptions compound. Local model inference requires always-on hardware. Rather than buying dedicated NAS hardware, I repurposed an old laptop into a headless server — solving both problems with hardware that was otherwise collecting dust.

Stack

  • OS: Ubuntu Server 24.04 LTS — headless, minimal footprint.
  • Management: CasaOS for Docker-based app deployment and monitoring.
  • Storage: Samba shares accessible across all devices on the local network.
  • AI: Ollama configured for local model hosting — inference without API costs.

Role: Sole builder

Stack: Ubuntu Server, CasaOS, Docker, Samba, Ollama

Status: Shipped & running
