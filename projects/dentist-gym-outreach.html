Dental Revenue Engine.

100+ personalised clinic websites deployed for ~₹91 total. Fully autonomous outreach.

Overview

A fully automated outreach system targeting independent dental clinics in India that lacked a web presence. Each clinic received a personalised, conversion-focused 1-page website at ₹9,999, with live previews deployed on GitHub Pages before the first email was even sent.

The Workflow

  1. Lead Generation — AI browser + Exa Neural Search to find 100+ qualified dental clinics (Rating 4.0+, aggregator-only listings, no independent site).
  2. Preview Build — Generated personalised, mobile-first Tailwind CSS website previews per clinic, deployed live on GitHub Pages.
  3. Outreach — Custom email sequences via Himalaya CLI with unique preview links per dentist.
  4. Monitoring — Autonomous email forwarding rule with Telegram and iMessage alerts on reply.

Coverage

Pune (Baner, Aundh, Wakad, Balewadi, Kothrud, Kondhwa), Mumbai (Malad, Ghatkopar, Khar, Juhu, Santacruz, Vile Parle), Bengaluru, and a national batch of 100 clinics.
