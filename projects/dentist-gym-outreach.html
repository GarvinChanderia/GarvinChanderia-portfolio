Dental Revenue Engine.

A fully autonomous outreach system that generated 100+ personalised clinic websites and deployed them live — total infrastructure cost under ₹91.

Thesis

Thousands of independent dental clinics in Indian metro areas have no web presence beyond a Google Maps listing. A conversion-focused, mobile-first single-page site is a high-value deliverable that can be generated programmatically and offered at a price point (₹9,999) that makes the ROI obvious. The question was whether the entire pipeline — discovery, generation, deployment, outreach, monitoring — could run without human intervention.

Pipeline

  1. Lead Discovery — Exa Neural Search + AI browser identifying clinics rated 4.0+ with no independent website. Filters: aggregator-only listings, no existing domain.
  2. Preview Generation — Tailwind CSS templates personalised per clinic (name, area, phone), deployed as individual folders on GitHub Pages. Each clinic sees their own live URL before any outreach occurs.
  3. Outreach Sequencing — Custom email sequences via Himalaya CLI. Each message includes the clinic's live preview link — the value proposition is visible before they reply.
  4. Reply Monitoring — Forwarding rules trigger Telegram and iMessage alerts on any response. Zero-latency human handoff for closing.

Coverage

Pune (Baner, Aundh, Wakad, Balewadi, Kothrud, Kondhwa), Mumbai (Malad, Ghatkopar, Khar, Juhu, Santacruz, Vile Parle), Bengaluru, and a national batch of 100 clinics across tier-1 cities.
