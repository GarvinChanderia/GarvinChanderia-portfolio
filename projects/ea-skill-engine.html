AI & Architecture

EA Skill Engine.

Autonomous capability mapping using LLMs to align IT stacks with business strategy.

The Problem

Enterprise Architecture practices rely on slow, manual knowledge gathering. Consultants spend weeks interviewing stakeholders and filling spreadsheets before they can produce a single capability map. The EA Skill Engine automates this — using structured LLM prompts to build, validate, and extend EA repositories at machine speed.

How It Works

The engine is built as a layered Claude skill system. A base EA skill provides methodology and metamodel knowledge. On top of that, tool-specific skills (like the Avolution ABACUS skill) handle platform integration. Organisation-specific skills (like the Bajaj Auto EA skill) encode client knowledge — coded element IDs, traceability chains, and domain relationships.

Architecture

The skill system follows a two-layer routing architecture: a generic reasoning layer that understands EA frameworks (TOGAF, ArchiMate, Zachman) and a platform-specific execution layer that generates ABACUS-compatible artifacts — VB.NET scripts, catalog configurations, and dashboard definitions.

Impact

Deployed in production at Invecto Technologies for the Bajaj Auto engagement. Used for KTM deal assessment, application portfolio rationalisation, and strategic architecture analysis — reducing weeks of manual work to hours of guided AI-driven sessions.
